# Expense Tracker

## Setup

npm install
npm run dev

## Features

- Add expenses
- Edit expenses
- Delete expenses
- Filter by category/title/date
- Monthly summary
- localStorage persistence

## Tech Stack

- React
- Vite
- localStorage

## Tradeoffs

Used localStorage instead of backend/database to prioritize shipping a stable working product within the given time constraints.

- Unit testing
